# Day 7 – Quantum Randomness Laboratory ⚛️

## Overview

On Day 7 of my **#30DaysProjectQ** journey, I explored the concept of **Quantum Random Number Generation (QRNG)** using **Qiskit**.

This hands-on lab demonstrates how a qubit in **superposition** produces truly random measurement outcomes, unlike classical pseudo-random number generators.


## Topics Covered

- Quantum Superposition
- Quantum Measurement
- Quantum Random Number Generation (QRNG)
- Histogram Visualization
- Statistical Analysis
- Classical vs Quantum Randomness


## Files

| File | Description |
|------|-------------|
| `qrng.py` | Generates quantum random numbers |
| `histogram.py` | Displays measurement histogram |
| `statistics.py` | Performs statistical analysis |
| `comparison.py` | Compares classical and quantum randomness |


## Technologies Used

- Python
- Qiskit
- Qiskit Aer
- NumPy
- Matplotlib


## Key Learning

Through this lab, I learned how quantum superposition enables probabilistic outcomes and how quantum-generated randomness differs from algorithm-based classical randomness.
